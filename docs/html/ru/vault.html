Vault — Хранилище секретов

Единственный модуль, знающий plaintext API-ключи провайдеров

RU EN

Назначение

Хранилище секретов с паттерном Sign-not-Expose. API-ключи шифруются AES-256-GCM per-record и никогда не покидают Vault в открытом виде.

Vault — единственная точка, через которую происходит доступ к API-ключам провайдеров. Ни один другой модуль не имеет доступа к plaintext значениям ключей.

Хранение

SQLite + AES-256-GCM

Хранилище построено на modernc.org/sqlite (pure Go, без CGO). Каждая запись шифруется отдельно с помощью AES-256-GCM. Формат хранения:

nonce (12 байт) || ciphertext

Per-record шифрование позволяет атомарно обновлять отдельные ключи без перешифрования всей базы.

KDF

Argon2id

Параметры ключевой функции:

ПараметрЗначение
Итерации3

Потоки4

Соль хранится открыто в metadata. Производный ключ — только в RAM (mlock-pinned буфер).

Sign-not-Expose

Custom RoundTripper

Паттерн Sign-not-Expose реализован через custom http.RoundTripper, который инжектирует Authorization header на уровне transport и зануляет его после записи в wire.

Plaintext ключ не задерживается в http.Request.Header. Вызывающий код никогда не получает доступ к значению ключа — только к подписанному запросу.

Токены

Аутентификация проектов

ValidateToken() использует HMAC(token) для lookup в БД + constant-time compare. Токены хранятся как HMAC хеши — plaintext никогда в БД.

TTL настраивается per-project. При истечении TTL токен автоматически инвалидируется.

Ротация

Атомарная замена ключей

RotateProviderKey() атомарно заменяет ключ в транзакции. Старый ключ немедленно удаляется и зануляется. Во время ротации запросы к провайдеру блокируются до завершения транзакции.

Безопасность памяти

explicit_bzero() — best-effort в Go. GC может скопировать данные до зануления.

Реальная граница безопасности — process isolation + mlock. Prism биндится только на 127.0.0.1, а mTLS обеспечивает защиту на уровне процессов.

Интерфейс

type Vault interface {
    SignRequest(ctx context.Context, projectID string, providerID types.ProviderID, req *http.Request) error
    RegisterProject(projectID string, allowedProviders []types.ProviderID, tokenTTL time.Duration) (token string, err error)
    ValidateToken(token string) (projectID string, err error)
    RevokeToken(token string) error
    AddProvider(providerID types.ProviderID, apiKey string, allowedProjects []string) error
    RotateProviderKey(providerID types.ProviderID, newAPIKey string) error
    RemoveProvider(providerID types.ProviderID) error
    Close() error
}
