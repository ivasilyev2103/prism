Provider — Адаптеры провайдеров

Нормализация различных AI API к единому интерфейсу Provider

RU EN

Назначение

Pass-through прокси к AI-провайдерам. Получает SanitizedBody и проксирует к API с минимальными изменениями (auth header, endpoint rewrite).

Provider не модифицирует содержимое запроса — только добавляет аутентификацию через Vault и формирует правильный URL endpoint.

Реализации

Провайдер ServiceTypes BillingType Статус
Claude chat per_token Полный
Ollama chat, embedding, image бесплатный Полный
OpenAI chat, image, embedding, tts, stt, moderation per_token, per_image, per_request, per_second Stub
Gemini chat, embedding per_token Stub

Execute()

Формирует URL endpoint провайдера, добавляет auth header через Vault.SignRequest(), проксирует SanitizedBody и парсит ответ в types.Response.

Контекст должен поддерживать отмену (timeout). При ошибке провайдера возвращается ошибка с HTTP-статусом и телом ответа для диагностики.

EstimateCost()

Предварительная оценка стоимости для Budget Guard. Выбирает BillingType по ServiceType и параметрам запроса.

Для per_token оценка основана на количестве входных токенов и модели. Для per_image — на количестве запрашиваемых изображений. Для Ollama всегда возвращает нулевую стоимость.

SupportedServices()

Объявляет поддерживаемые ServiceType для проверки совместимости роутером. Router использует эту информацию, чтобы не направлять image-запросы к провайдеру, поддерживающему только chat.

Registry

Реестр провайдеров

МетодОписание
Get(id)Возвращает провайдер по ID. Ошибка если не зарегистрирован.
GetForService(serviceType)Возвращает провайдеры, поддерживающие указанный тип сервиса.
Register(p)Добавляет провайдер. Паникует при дублировании ID.
All()Возвращает все зарегистрированные провайдеры.

Интерфейс

type Provider interface {
    ID() types.ProviderID
    SupportedServices() []types.ServiceType
    Execute(ctx context.Context, req *types.SanitizedRequest) (*types.Response, error)
    EstimateCost(req *types.ParsedRequest) (*types.CostEstimate, error)
    HealthCheck(ctx context.Context) error
}

type Registry interface {
    Get(id types.ProviderID) (Provider, error)
    GetForService(serviceType types.ServiceType) []Provider
    Register(p Provider)
    All() []Provider
}
