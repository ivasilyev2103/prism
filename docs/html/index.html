Prism

LocalAI Gateway — privacy-first, cost-controlled, smart-routed AI proxy

🔒

Privacy-First

PII obfuscated before reaching the cloud. Tiered detection: Regex, Ollama NER, Presidio.

💰

Cost Control

4-level budget hierarchy. Per-token, per-image, subscription billing. Real-time tracking.

🔌

Smart Routing

Rule-based routing with ServiceType awareness. Automatic failover between providers.
